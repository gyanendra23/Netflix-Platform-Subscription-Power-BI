# Netflix-Platform-Subscription-Power-BI
This project provides a complete end-to-end analysis of Netflix subscription data for the year 2022. It focuses on understanding subscriber behavior, monthly revenue patterns, churn trends, and platform growth, all visualized through an interactive Power BI dashboard.
Project Overview

The objective of this project is to explore how subscription-based platforms like Netflix track performance KPIs such as revenue, churn, retention, and new subscriber acquisition.

Using real-world structured data, the dashboard highlights:

Subscription lifecycle trends

Monthly & quarterly revenue performance

User activity vs inactivity

Seasonal fluctuations

Growth and churn drivers
